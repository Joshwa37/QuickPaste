# 🚀 QuickPaste – Text Sharing Platform

QuickPaste is a simple and fast web application that allows users to share text snippets instantly using unique links — inspired by JustPasteIt.

---

## 📌 Features

* ✨ Create and share text instantly
* 🔗 Unique URL for each paste
* ⚡ Fast retrieval of saved content
* 🗂️ Backend storage with database integration
* 📱 Clean and responsive UI
* 🔒 Optional support for secure/private pastes (if implemented)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **API:** RESTful API

---

## ⚙️ How It Works

1. User enters text in the input box
2. Clicks on **Create Paste**
3. Backend stores the text and generates a unique ID
4. A shareable link is created
5. Anyone with the link can view the paste

---


## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/quickpaste.git
cd quickpaste
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Server

```bash
npm start
```

### 4. Open in Browser

```
http://localhost:3000
```
